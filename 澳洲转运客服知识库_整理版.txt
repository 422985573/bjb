澳洲转运客服知识库（整理版）

========================================

- 货量较多时，可能到周二/周三才全部入库完成
- 假期堆积时，入库时间会延长
- 包裹入库后，公众号会有通知

1.3 悉尼仓库工作时间
- 地址：Gate 1 / Unit 8 / 40 Brodie St, Rydalmere NSW 2116
- 工作时间：周一至周五 10:00-17:00
- 午休时间：12:00-12:30
- 请于17:00前完成自提

1.4 墨尔本仓库工作时间
- 地址：Building J, 413 Francis Street, Brooklyn VIC 3012
- 工作时间：周一至周五 10:00-17:00（澳洲时间）
- 最后自提时间：17:00前

2. 服务范围
========================================

2.1 业务范围
- 只接中国寄往澳大利亚
- 只做澳洲空运 / 海运
- 不接其他国家地区
- 不提供澳洲寄回中国服务
- 不提供其他国家转运服务

3. 仓库地址获取与注册流程
========================================

3.1 深圳仓库地址
- 深圳仓库地址需按流程获取
- 操作方式：
  1. 关注公众号：海山国际物流
  2. 完成注册
  3. 在群里发送会员号 / ID
  4. 客服协助修改地址格式并提供仓库地址

- 新客户：按注册流程完成注册后，把ID发群里
- 老客户：登录公众号后，核对会员号是否与之前一致
- 如不一致，需联系客服处理

========================================
4. 下单、预报、打包流程
========================================

4.1 快递预报流程
公众号进入“会员中心” → “首页” → “快递预报”

操作说明：
1. 支持批量预报
2. 正确填写快递单号
3. 数量和货值可先填写
4. 建议在“品名”处写清物品名称，方便后续打包识别

4.2 申请打包流程
所有包裹到齐后：
1. 进入公众号 → 我的包裹
2. 勾选本次需打包的快递单号
3. 点击右下角“申请打包”
4. 填写收件地址等信息（需英文格式填写）

4.3 下单注意事项
- 一个订单默认打包发货一次
- 提交订单后不支持分货
- 若申请后再加减包裹，会收取50 RMB/票操作费
- 提交订单前务必核对好物品

4.4 扣件 / 修改限制
以下情况不支持扣件或修改信息：
- 包裹已出仓
- 已装柜
- 已交仓
- 已安排航班 / 船期

可扣件情形：
- 包裹打包好后
- 尚未支付运费
- 尚未安排船期 / 航班

========================================
5. 打包规则
========================================

5.1 默认打包规则
- 默认拆除外包装
- 如需保留包装，需提前联系客服说明
- 易碎品 / 家用电器：仓库一般会保留原包装发货
- 收藏品类（小卡、周边、谷子、徽章、手办、盲盒、捏捏等）默认保留飞机盒打包

5.2 特殊打包备注
- 可在公众号申请打包时，在备注栏填写要求
- 如：
  - 包装方式
  - 易碎提醒
  - 拆包 / 保留包装
  - 加固要求
- 同时建议在群里同步备注一次

5.3 小件标准箱
- 小件默认打包箱尺寸：50 × 40 × 30 cm
- 打包后尺寸会有轻微浮动

5.4 6邮编（珀斯）包裹规格
- 6邮编零散小件需使用规定纸箱：50 × 40 × 30 cm
- 实重不超20kg
- 不接：
  - 单边 > 50cm
  - 实重 > 20kg
  - 超长超大件

========================================
7. 渠道与时效
========================================

7.1 时效标准回复（必须严格按以下格式回复）

空运时效：航班起飞后约7-11个工作日签收，（不遇到航司爆仓甩货，二程延误，海关查验，恶劣天气等不可抗力因素）具体以物品到达时间为准。

海运实效：海运每周装柜1-3次，装柜后等船司安排船期。开船后约35-45天可以签收（不含船司甩柜、海关查验、大船晚靠、目的港塞港等不可抗力因素）具体以物品到达时间为准。

特快空运时效：空普特快当天或隔天交货到机场起飞，约3-5个工作日左右签收(不含海关查验延误等特殊情况)具体以物品到达时间为准。

珀斯海运时效：正常情况下海运每周装柜1-3次，装柜后等船司安排船期。开船后约45-55天签收(不含船司甩柜，海关查验，大船晚靠，目的港塞港等不可抗力因素)具体以物品到达时间为准。

珀斯空运时效：航班起飞后约10-15个工作日可以签收，(不遇到航司爆仓甩货，二程延误，海关查验，恶劣天气等不可抗力因素)具体以物品到达时间为准。

7.2 海运装柜 / 开船
- 正常情况下海运每周装柜1-3次
- 装柜后等待船司安排船期
- 物流轨迹中会更新预计开船时间

7.3 空运交仓 / 起飞
- 空运按航班表交仓
- 每周交仓时间不固定
- 交仓后，如无查验等情况，预计3-5天左右起飞

7.4 靠港后派送
- 靠港后预计7-11个工作日左右送达
- 中间时间用于：
  - 码头提柜
  - 分拣
  - 交付快递

8. 价格与计费基础
========================================

8.1 最低起运

8.2 参考价格
- 空运约：45元/kg
- 海运约：18元/kg
- 实际费用需根据：

  - 重量
  - 邮编

8.3 邮编报价规则
- 6邮编开头 = 珀斯价格表
- 询问墨尔本 / 悉尼 / 布里斯班 / 阿德莱德收费：发送空运价格表 + 海运价格表
- 询问珀斯收费：直接发送珀斯价格表

8.4 立方报价
- 立方报价需咨询人工客服

========================================
9. 体积重 / 材积重规则
========================================

9.1 定义
- 材积重 = 体积重

9.2 公式
- 海运材积重：长 × 宽 × 高 ÷ 6000
- 空运材积重：长 × 宽 × 高 ÷ 5000
- 1立方米（CBM）= 167KG

9.3 计费原则
- 空运、海运均按体积重与实际重取大值计费

9.4 测量规则
1. 方正包裹：按纸箱长宽高计算
2. 曲面包裹：按最凸点计算
3. 不规则包裹：每边按最长边计算体积重

9.5 海运自提特殊规则
在以下条件内：
- 单边不超100cm
- 三边和不超180cm
- 实重不超22kg

则：
- 按实重计费
- 不计算体积重
- 无附加费

超出任一条件：
- 按实重 / 体积重取大值
- 但没有附加费

========================================
10. 支付与税费
========================================

10.1 支付方式
支持：
- 微信
- 支付宝
- 淘宝
- 澳洲银行账户转账

10.2 GST
- 转澳洲银行账户需加收10% GST

10.3 税金规则
- 单次邮寄总货值低于1000澳币：通常不产生税金，走不含税渠道
- 达到1000澳币及以上：
  - 需走包税渠道
  - 或分批转运
  - 也可能产生税金及杂费

10.4 高货值杂费
- 单次邮寄总货值达到1000澳币，除税金外，可能还有100-200澳币海关系统费等杂费
- 具体以实际账单为准

========================================
11. 包税 / 不包税规则
========================================

- 货值低于1000澳币：通常走不含税渠道
- 货值达到1000澳币：需走包税渠道或分批次转运
- 奢侈品不能走包税渠道
- 全是敏感货无法走包税渠道
- 只有“大量普货 + 少量敏感货”才有机会走包税渠道

========================================
12. 保险与赔付
========================================

12.1 保险规则
- 保费：保价金额的3%
- 最低保价：1700元
- 最高保价：10000元
- 最低消费：50元/票
- 保险按整箱包裹投保
- 仅适用于可投保普货
- 敏感货 / 旧物 / 大牌 / 收藏品等不可投保
- 只保丢，不保损

12.2 投保赔付规则
- 按实际货值与投保货值取低赔付
- 可退运费
- 需提供购物截图等凭证

12.3 不购买保险
- 默认赠送：丢失险700 RMB + 退运费
- 赔付按实际货值与700元取低
- 同时退运费

12.4 不适用保险 / 不赔场景
以下情况不适用：
- 海关没收
- 海关销毁
- 小偷偷件
- 签收后丢件
- 敏感货
- 旧物
- 大牌
- 收藏品类
- 易碎品破损

========================================
13. 统一赔付标准
========================================

1. 运输中破损：最高赔付100元/箱
2. 易碎品（如塑料、玻璃）：不赔
3. 食品、药品等敏感货被海关查扣：不赔
4. 国内丢件：赔付包裹货值
5. 澳洲末端快递提取后丢件：最高不超700元/箱 + 退运费
6. 签收后丢件：不赔
7. 保价包裹丢失：按实际货值与保价金额取低赔付
8. 售后需在签收后72小时内提供：
   - 开箱视频
   - 运单号
   - 快递单号
   - 购物支付截图
   否则不受理

========================================
14. 末端派送与退回
========================================

14.1 末端快递
- 由 AusPost 或 Toll 派送

14.2 派送异常
如出现以下情况：
- 地址不详细
- 无人签收
- 派送失败

可能结果：
- AusPost：放到附近邮局
- Toll：带回仓库
- 也可能退回悉尼仓库

14.3 重派费用
以下情况会产生二次派送费：
- 包裹在邮局放置太久被退回悉尼仓库
- Toll带回仓库后重新派送
- 客户主动联系重新派送

14.4 邮局取件要求
- 需携带收件人实名护照
- 需带电子面单
- 需在邮局营业时间内取件
- 避免超时被退回悉尼仓库

14.5 海外退件 / 澳洲退件
- 多因地址问题或派送失败
- 包裹会暂放邮局 / 仓库
- 超时未取会退回悉尼仓库
- 退回后可选择：
  - 悉尼仓库自提
  - 重新派送（需付二次派送费）

========================================
16. 品类定义
========================================

16.1 普货
不带以下属性的普通物品：
- 不带磁
- 不带电
- 不带液体
- 无安全隐患

16.2 敏货
介于普货和违禁品之间，一般包括：
- 带磁
- 带电
- 带液
- 膏体
- 粉末
- 食品
- 药品
- 化妆品
- 品牌货
- IP形象物品等

特点：
- 可邮寄的部分物品存在海关查扣风险
- 查扣不赔

16.3 奢侈品
- 可邮寄
- 不能走包税渠道
- 不能参保
- 丢件、破损一般不赔

16.4 电子产品
- 原则：不建议邮寄 / 多数不接
- 包括：
  - 手机
  - 电脑
  - 平板
  - 耳机
  - 主机
  - 相机
  - CCD
  - 音响等
- 无法投保
- 丢件、破损不赔

========================================
17. 可邮寄物品
========================================

可邮寄大类：
- 各类普货
- 潮牌
- 食品（注意敏感属性）
- 奢侈品（接受条款）
- 少量正规商业包装液体（特定类别）
- 少量正规商业包装小电池 / 充电宝（满足条件）
- 酒类（仅海运、正规商业包装）

========================================
19. 特殊品类规则
========================================

19.1 药品
- 处方药、消炎药、精神类药物：不接
- 普通药品：属于敏货
- 有查扣风险，查扣不赔

19.2 液体
- 只可少量邮寄：
  - 洗衣液
  - 调味品
- 奶制品、饮料：不能邮寄
- 液体属于敏货，查扣不赔

19.3 酒类
- 只能走海运
- 自营空运 / 特快空运均不接
- 需有正规商业包装
- 易拉罐不接
- 属于敏货 + 易碎品
- 海关扣件不赔
- 运输破损最高赔100元/箱

19.4 电池 / 充电宝
- 少量有正规商业包装的：
  - 纽扣电池
  - 5号电池
  - 充电宝（单个及总容量低于20000mAh）
  可搭配普货邮寄
- 超过20000mAh、户外电源、大电池：
  - 需走DG柜（海运）
  - 在基础报价上 +5元/kg
- 充电宝需有3C认证

19.5 家具
- 需提供：
  - 三边尺寸
  - 实重
  - 详细地址
- 才能确认是否可邮寄
- 派送可能涉及：
  - 超长超重费
  - 尾板卸货费
  - 私人住址费

21. 海关与行业术语
========================================

21.1 海关查验
- 分为：
  - 国内海关查验
  - 澳洲海关查验
- 空运、海运都会经过X光扫描
- 海关系统随机布控，抽查概率较低
- 查验时间以海关通知为准

21.2 查验类型
- 普通开箱查验：随机抽取包裹查验
- AQIS查验：查品名、材质、申报一致性
- EMPP查验：查申报货值与实际货值是否一致，可能需要：
  - 购买链接
  - 购买截图
  - 付款水单

21.3 海关扣件 / 扣货
- 一般只查扣敏货
- 普货通常放行
- 不会整箱都扣，但可能影响同箱其他货物
- 敏货查扣不赔

21.4 术语解释
- 大船晚靠：晚于预定时间靠港
- 飞机二程延误：中转后未衔接上下一程航班
- 飞机爆仓：货量超舱位，部分货物顺延
- 塞港：港口拥堵，船舶排队靠泊
- 甩柜：船司超量放舱，货柜顺延下一班船

24. 统一客服口径提醒（容易答错的地方）
========================================

下面这些是从原文里找出来的高风险答错点，建议单独做成内部注意事项。

24.1 电子产品口径
原文里同时出现了：
- “电子产品不能邮寄”
- 但又有“小电池、充电宝可少量搭配普货邮寄”

建议内部统一为：
- 大多数电子产品不接
- 少量符合要求的小电池 / 充电宝需单独按规则处理
- 客服不要简单答“所有带电都可以”或“所有带电都不行”

24.2 奢侈品口径
原文里有两种说法：
- “奢侈品可邮寄但扣货不赔，并建议走包税渠道”
- “奢侈品不能走包税渠道”
- “奢侈品说明：不能走包税渠道”

这里明显有冲突。
建议最终以这条为准：奢侈品不能走包税渠道。
因为后文重复出现了两次：
- 奢侈品无法走包税渠道
- 奢侈品不能走包税渠道

24.3 敏货渠道口径
原文里有：
- “我们家没有敏货渠道”
- 但又说“空运海运大部分敏感货是可以和普货一起邮寄的”

建议统一理解为：
- 没有单独命名的“敏货专线”
- 但空运/海运普通渠道内，可接部分敏感货
- 特快空运不接敏货

24.4 仓库垃圾清理费币种
